# label-print-topbridge-js-core-demo

[@appzgatenz/label-print-topbridge-js](https://www.npmjs.com/package/@appzgatenz/label-print-topbridge-js) SDK 的开发者文档和交互式示例。

## 本地开发

```bash
pnpm install
pnpm dev
```

浏览器打开 `http://localhost:5173`。

## 内容

- **指南** — 安装、初始化、配置、数据转换、错误处理、故障排查
- **示例** — 可交互的 HTML 示例，覆盖核心 API 调用场景
